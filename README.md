# 🎁 Customer Sales Analysis & Dashboard
## 🚀 Excel-Based Business Intelligence Project

---

## 📖 Project Overview

This project provides a complete analysis of customer purchasing behavior, revenue trends, and product performance for a gifting retail business.

By combining multiple datasets (orders, customers, and products), the project uncovers key business drivers across:

- 🎉 **Occasions**
- 📍 **Locations**
- ⏰ **Time Patterns**

💡 The final output is an **interactive Excel dashboard** designed for business decision-making.

---

## 🎯 Business Objectives

- 📈 Analyze revenue and order trends
- 🏆 Identify top-performing products and categories
- 👥 Understand customer behavior across demographics
- 🗺️ Discover location-based and time-based sales patterns

---

## 🗄️ Data Architecture

The project integrates **3 datasets:**

| Dataset | Details |
| ------------------ | ----------------------------------------- |
| 📄 Orders Dataset | 1,000 transactions (date, time, location, occasion) |
| 👥 Customer Dataset | 100 unique customers (gender, city, demographics) |
| 🛍️ Products Dataset | 70 products (category, price in INR) |
| 📊 Excel Dashboard | Final integrated dataset + interactive dashboard |

## 🔍 Key Insights

### 📈 Revenue Drivers
- 🎨 **Top Category:** Colors (highest revenue contributor)
- 🧸 Followed by **Soft Toys** and **Sweets**
- 🎉 **Peak Occasions:** Anniversary & Raksha Bandhan

### 👥 Customer Insights
- Gender contribution is nearly balanced:
  - 👩 Female: **51%**
  - 👨 Male: **49%**

### 🛍️ Product Performance
- Top-selling products include:
  - 🥇 Magnam Set
  - 🥈 Quia Gift
  - 🥉 Dolores Gift

### 📍 Location Insights
- Highest revenue cities:
  - 🏙️ Vijayanagram
  - 🏙️ South Dumdum
  - 🏙️ Ghaziabad

### ⏰ Time-Based Trends
- Sales peak during:
  - 🌅 **16:00** (Afternoon)
  - 🌙 **20:00** (Evening)

## 🛠️ Technical Implementation

### 🧹 Data Preparation
- Cleaned and standardized date & time formats
- Unified categorical values across datasets

### 🔗 Data Modeling
- Linked datasets using:
  - `Customer_ID`
  - `Product_ID`

### 📊 Dashboard Features

| Feature | Description |
| --------------- | --------------------------------------- |
| 🎛️ Slicers | Filter by Date, Occasion, Delivery timeline |
| 📈 Pivot Charts | Revenue by category, location, and time |
| 🍩 Donut Charts | Gender-based revenue distribution |

## 🚀 How to Use

   - 📅 Months (2023–2024)
   - 🎉 Occasions
   - 🚚 Delivery timeline
4. Explore revenue trends dynamically
